# Лицензирование FlowScope (RU)

## Выбранная лицензия

Проект распространяется под лицензией:

- `GNU Affero General Public License v3.0 or later`
- SPDX: `AGPL-3.0-or-later`

## Где это зафиксировано

- Полный текст лицензии: `LICENSE`
- Уведомление о правообладателе: `NOTICE`
- Указание лицензии в frontend package: `web/package.json`

## Что это означает на практике

1. Можно использовать, модифицировать и распространять проект на условиях AGPL.
2. При распространении изменений исходный код должен быть доступен на условиях AGPL.
3. Для сетевого использования (SaaS-сценарий) AGPL также накладывает обязательства по предоставлению исходного кода модификаций.

## Рекомендации для GitHub

1. В `About` оставить лицензию `AGPL-3.0`.
2. В релизах и документации явно указывать SPDX: `AGPL-3.0-or-later`.
3. Для сторонних вкладов использовать CLA/DCO только при необходимости вашей юридической политики.
